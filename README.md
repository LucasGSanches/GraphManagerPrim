# GraphManagerPrim



Código base: código do Grassi mostrado em aula

Implementação necessária: algoritmo gerador de árvore mínima



Progresso feito: já foi declarado a função e suas variáveis, basta implementar o algoritmo Prim e colocar as arestas constituintes do grafo gerado na variável "arestasTemp" que refletirá na interface automaticamente. A variável "pontoTemp" só serve para o próprio algoritmo funcionar, já que todos os vértices devem ser mantidos, então a alteração dele não reflete no grafo.
