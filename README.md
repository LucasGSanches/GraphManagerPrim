# GraphManagerPrim



Código base: código do Grassi mostrado em aula

Implementação necessária: algoritmo gerador de árvore mínima



Estratégia planejada por Lucas: 

Uma função que vai receber o grafo, calcular o grafo mínimo e substituir o grafo original pelo grafo mínimo na interface.

Para isso a função vai ficar escolhendo a aresta com o menor peso e ficar verificando se o vértice não já é integrante do grafo que está sendo feito, e se for o caso não vai adicionar essa aresta e vai verificar o próximo até que todos os vértices estejam conectado.
